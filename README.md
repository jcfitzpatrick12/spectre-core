## Installation
Simply call ```pip install spectre_core```. Alternatively, for development you can clone the repository in your preferred directory:  
```git clone https://github.com/jcfitzpatrick12/spectre-core.git```  

Change directory:  
```cd spectre-core```  

Then run:  
```pip install -e .```
